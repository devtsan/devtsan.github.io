BTGWAB: Blog Template for GhPage using WordPress.com As Backend

WordPress siteမှာ တင်သမျှ postတွေ ကိုယ့်ရဲ့ GhPage (GitHub Pages) မှာ ပေါ်နေဖို့ template တခုပါ

ဒီ version မှာ WordPress.com free planကို သုံးထားတဲ့ WordPress siteက dataကို GhPage ဆီ integrate လုပ်ပေးထားပါတယ်၊ အသုံးပြုပုံကိုတော့ Documentation pageမှာ ကြည့်ပါ။

Current version: 1.0 (Alpha release)

ပထမဆုံး release တဲ့ version ဖြစ်လို့ feature တွေ မစုံလင်သေးပါဘူး၊ theme အသစ်တွေလည်းထပ်ရေးပေးဖို့ရှိပါတယ်၊ ခု alpha release မှာတော့ default appearance တခုသာ ရွေးချယ်စရာရှိပါတယ်။ theme & feature request လိုရင် pull request လုပ်ပြီး feedback ပေးလို့ရပါတယ်။

Self-hosted siteတွေအတွက် WordPress backend နဲ့ headless CMS ကိုလည်း မကြာခင်ထုတ်ပေးဖို့ရှိပါတယ်၊ BTGWAB မှာတော့ monetization support မပါဝင်ပါဘူး၊ self-hosted version မှာတော့ monetization support ပါဝင်လာပါမယ်။

BTGWAB နဲ့ ပါသက်ပြီး အကူညီလိုအပ်ရင် Develope's Facebook Page ရဲ့ message ကတဆင့် အကူညီရယူနိုင်ပါတယ်။
